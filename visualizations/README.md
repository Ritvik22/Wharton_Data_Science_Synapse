# WHL 2025 Phase 1a — Visualizations

Charts are generated by running from the project root:

```bash
python3 visualize.py
```

| File | Description |
|------|-------------|
| **01_league_standings.png** | Season points for all 32 teams (league table order). Top 8 in blue. |
| **02_power_rankings.png** | Elo power rating for all 32 teams. Top 8 in dark blue. |
| **03_league_vs_power_rank.png** | League rank vs Power rank. Points above the diagonal = stronger schedule; below = weaker schedule. |
| **04_first_round_win_probabilities.png** | Home team win probability (%) for each of the 16 first-round tournament matchups. |
| **05_goal_differential.png** | Season goal differential (GF − GA) by team. Green = positive, red = negative. |
| **06_summary_dashboard.png** | 2×2 summary: points distribution, Elo distribution, top-8 win probs, league vs power rank. |
